# Django project

Init venv

```shell
~$ python -m venv ./venv
```

Activate venv

```shell
~$ .\env\Scripts\activate.ps1
```

Download Django

```shell
~$ pip install django
```

Create project

```shell
~$ django-admin startproject initproject
```

Create application

```shell
~$ python .\manage.py startapp challenges
```

Run project

```shell
~$ python .\manage.py runserver
```
